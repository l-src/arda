Website schreiben

Die erste Sache die einen erstaunt ist, dass man um zu starten, keinerlei neue Softwares auf dem PC benötigt.
Man kann breits mit den Tools, die man auf dem Pc hat, loslegen.

Wenn man Windows hat, muss man einfach nur unten auf das Start-Menü klicken, auf die Suche gehen und nach dem
"Editor" suchen. Der Editor ist ein simples Programm, wo man einfach Text reinschreiben kann.

Wenn man Mac-Nutzer ist, sollte man TextEdit benutzen. Anschließend muss man in TextEdit noch ein Paar kleine Änderungen vornehmen.
Man muss oben auf Format drücken und dann die Option Make Plain Text auswählen.

Nun kann man bereits seine erste HTML Datei schreiben
Das heißt, man kann jetzt irgendeinen Text schreiben ( Beispiel : "Meine erste Website") und anschließend die Datei im Explorer speichern
(am besten im Desktop). Am besten legt man zuerst einen Ordner für die Internetseite an und speichert diese in diesem Ordner.
Man sollte zuvor nicht vergessen bei "Dateityp:" "Alle Dateien" auszuwählen und der Datei den Namen "index.html" zu geben.
(Bei Mac ist das genau so)

Wenn man jetzt den Explorer öffnet, sieht man dort die HTML Datei, welche man jetzt mit dem Webbbrowser öffnen kann.
Nötig ist dafür nur ein Doppel-Klick auf die Datei.

Was genau ist HTML?

Bevor es richtig losgeht, sollte man sich erst ein mal damit beschäftigen was HTML genau ist.
HTML ist eine Markup Language. Das bedeutet es ist eine Sprache mit denen Informationen dargestellt
werden. Es ist keine Programmiersprache, aber trotzdem ist es, wenn man Programmieren lernen möchte,
der ideale Einstieg.

Man kann mit HTML angeben, welche Informationen sich auf der Seite befinden sollen.
Beispiele :
- Überschrift
- Absatz mit Text
- Bild

Als aller erstes muss man angeben, welche Informationen in dieser Applikation dargestellt werden
sollen und das macht man mit HTML

Zusätzliche Informationen :
-Mit CSS kann man sagen wie das ganze Aussehen soll. Das heißt da kann man die Abstände zwischen
den Elementen definieren, Farben zuweisen, Größen bestimmen und so weiter und so fort
-Mit JavaScript erstellt man die Logik

Das heißt also man hat 3 verschiedene Bausteine:
HTML = Informationen
CSS = Aussehen
JavaScript = Logik
